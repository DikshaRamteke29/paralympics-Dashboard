# paralympics-Dashboard
1 Project Title

Paralympics Performance Analytics Dashboard using Power BI

2 Project Overview

The Paralympics Performance Analytics Dashboard is a data visualization and business intelligence project developed using Power BI. This dashboard provides detailed insights into Paralympics medal performances across countries, continents, seasons, and years.

The project transforms raw historical Paralympics data into meaningful visual reports and interactive dashboards. It enables users to analyze medal trends, compare country performances, understand continent-wise distributions, and identify top-performing nations in Paralympics history.

The dashboard is designed to help sports analysts, researchers, organizations, and users quickly explore Paralympics statistics through interactive charts, KPI cards, filters, and tables.

3 Tech Stack
Power BI – Data visualization and dashboard creation
Excel / CSV – Data source
DAX (Data Analysis Expressions) – Calculated measures and KPIs
Power Query – Data cleaning and transformation

4 Data Source
Possible sources:

Kaggle Dataset
Paralympics historical records
CSV/Excel files

5 Features / Highlights
KPI Cards – Shows Total Medals, Gold Medals, Countries, and Games Duration.
Example: 29K Total Medals, 11K Gold Medals.

Medal Trend Analysis – Line chart showing medal performance over years.
Example: Medal growth increased after the 1980s.

Continent-wise Distribution – Donut chart for medal share by continent.
Example: Europe contributes the highest medals.

Top 10 Countries Analysis – Bar chart ranking countries by medals.
Example: United States ranks among the top countries.

Interactive Filters – Filters for Season, Country, and Year.
Example: Selecting “Winter” updates all visuals automatically.

Detailed Medal Table – Displays Gold, Silver, Bronze, and Total Medals country-wise.
Dynamic Dashboard – All charts interact with each other for better analysis.
Professional UI Design – Clean dark-themed dashboard for better visualization and user experience.

6 Business Problem
Sports organizations and analysts need a better way to track and compare Paralympics performance across countries and years. Raw data is difficult to analyze manually.

This dashboard solves the problem by:
Providing quick insights into medal performance
Identifying top-performing countries
Analyzing historical trends
Supporting data-driven sports analysis and reporting
